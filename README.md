# Todo List

This is a simple Flutter Todo List app (Tiếng Việt UI). The README below documents the repository layout, key files, features, and how to run or extend the app.

## Structure

- **android/**, **ios/**, **macos/**, **linux/**, **windows/**, **web/**
	- Platform folders generated by Flutter.
- **lib/**
	- `main.dart` — ứng dụng chính, UI list, filters, checkbox handling, và persistence wiring.
	- `insert.dart` — helper để hiện Dialog / BottomSheet thêm công việc (`showAddTodoDialog`, `showAddTodoBottomSheet`).
	- `fix.dart` — helper sửa công việc (prefill) và trả về kết quả chỉnh sửa (`showEditDialog`, `showEditBottomSheet`, `showEditAndSave`).
	- `delete.dart` — helper xác nhận xóa (`confirmDelete`).
- **test/**
	- `widget_test.dart` — test mẫu tạo bởi Flutter.
- `pubspec.yaml` — dependencies (ví dụ `shared_preferences`).
- `analysis_options.yaml` — linter / analyzer config.

## Công nghệ & Thư viện

- Flutter (Dart)
- shared_preferences — lưu danh sách dưới dạng JSON (key: `todo_items_v1`).

## Tính năng chính

- Danh sách công việc (ListView) với layout cân đối.
- Thêm công việc qua Dialog hoặc BottomSheet (hiện mặc định Dialog).
- Bắt buộc phải chọn cả ngày và giờ khi thêm (validation có trong `lib/insert.dart`).
- Sửa công việc với giao diện prefill (có thể sửa text và date/time).
- Xóa công việc có dialog xác nhận.
- Lọc: `Tất cả` / `Chưa xong` / `Đã xong` (ChoiceChip).
- Checkbox để đổi trạng thái; nội dung có gạch ngang + mờ khi hoàn thành.
- Lưu trạng thái và danh sách (cùng dueDate) bằng `shared_preferences` (JSON).

## Key implementation notes

- Model: `TodoItem` (có `title`, `done`, `DateTime? dueDate`) — có `toJson()` / `fromJson()`.
- Persistence: `_saveData()` và `_loadData()` được gọi trong `lib/main.dart` — mọi thao tác thay đổi (thêm / sửa / xóa / toggle) gọi `_saveData()` để lưu.
- Date & Time: khi chọn ngày và giờ, code kết hợp thành một `DateTime` duy nhất trước khi lưu.
- Validation: `lib/insert.dart` hiện đã yêu cầu `text` không rỗng và `date` + `time` đều phải được chọn trước khi bật nút "Thêm".

## Chạy ứng dụng (local)

1. Cài Flutter và thiết lập môi trường theo tài liệu chính thức.
2. Từ thư mục gốc dự án chạy:

```bash
flutter pub get
flutter run
```

Nếu cần chạy cho nền tảng cụ thể (Android / iOS / Windows), thêm cờ hoặc chọn thiết bị tương ứng.

## Thay đổi cấu hình UI / hành vi

- Đổi mặc định Dialog vs BottomSheet khi thêm/sửa: kiểm tra nơi gọi `showAddTodoDialog` / `showAddTodoBottomSheet` và `showEditAndSave(..., useBottomSheet: true)` trong `lib/main.dart`.
- Thay đổi khoá persistence: tìm `_prefsKey` trong `lib/main.dart`.

## Gợi ý nâng cao (tùy chọn)

- Thêm `undo` khi xóa (SnackBar) để phục hồi item.
- Thêm `intl` để định dạng ngày/giờ theo locale.
- Bổ sung chế độ `Dark Mode` trong `ThemeData` và toggle trong Settings.
- Thay `shared_preferences` bằng DB (sqlite / sembast) nếu cần danh sách lớn hoặc truy vấn phức tạp.

## Liên hệ / phát triển tiếp

Nếu muốn, tôi có thể:
- Thêm `undo` cho xóa.
- Triển khai `Dark Mode`.
- Chạy kiểm thử cơ bản hoặc thử nghiệm `flutter run` và gửi kết quả.
